Datasets¶

There are four files associated with this project:

  • Udacity_AZDIAS_Subset.csv: Demographics data for the general population of Germany; 891211 persons (rows) x 85 features (columns).
  • Udacity_CUSTOMERS_Subset.csv: Demographics data for customers of a mail-order company; 191652 persons (rows) x 85 features (columns).
  • Data_Dictionary.md: Detailed information file about the features in the provided datasets.
  • AZDIAS_Feature_Summary.csv: Summary of feature attributes for demographics data; 85 features (rows) x 4 columns

Each row of the demographics files represents a single person, but also includes information outside of individuals, including information about their household, building, and neighborhood. I used this information to cluster the general population into groups with similar demographic properties.

Step 1: Preprocessing¶

Step 1.1: Assessing Missing Data by Converting Missing Value Codes to NaNs¶

I started the process of data preparation by converting the missing value codes to NaNs in the actual dataset. The codes for missing or unknown values were obtained from the document of detailed information on the features. After converting all the missing or unknown values in the dataset to NaN, I wanted to work on identifying the pattern of missing values in the dataset. First, I wanted to find how much missing data is present in each column, so I worked on finding the outliers columns, which are columns that are in different proportions in terms of values that are missing.

Investigate patterns in the amount of missing data in each column.¶

After visualizing the distribution of missing value, we can see that there are some features with missing data more than most of the attributes in the dataset. For this dataset, it seems that the outliers are the features that have above 200,000 missing data.

Based on the observation above, I made the decision of removing the following columns: ‘AGER_TYP’, ‘GEBURTSJAHR’, ‘TITEL_KZ’, ‘ALTER_HH’, ‘KK_KUNDENTYP’, and ‘KBA05_BAUMAX’, as they had above 200,000 missing data.

def clean_data(df):
    """
    Perform feature trimming, re-encoding, and engineering for demographics
    data
    
    INPUT: Demographics DataFrame
    OUTPUT: Trimmed and cleaned demographics DataFrame
    """
    azdias = df
    # Put in code here to execute all main cleaning steps:
    # convert missing value codes into NaNs, ...

    feat_info = pd.read_csv('AZDIAS_Feature_Summary.csv', delimiter=';')
    feature_list = feat_info[['attribute', 'missing_or_unknown']]
    parsed_feature_dict = {}
    import json
    for item in feature_list.values:
        res = item[1].strip("][").split(",")
        parsed_feature_dict[item[0]] = res

    for column in azdias.columns:
        for item in parsed_feature_dict[column]:
            if(item == ''):
                continue
            if azdias[column].dtype == 'object':
                azdias.loc[azdias[column] == item,column] = np.nan 
            else:
                azdias.loc[azdias[column] == int(item),column] = np.nan
    
    # remove selected columns and rows, ...
    outliers_list=['AGER_TYP', 'GEBURTSJAHR', 'TITEL_KZ', 'ALTER_HH', 'KK_KUNDENTYP', 'KBA05_BAUMAX'] #['AGER_TYP', 'TITEL_KZ', 'KK_KUNDENTYP', 'KBA05_BAUMAX']

    #Any attribute in outliers_list will be dropped from the dataset
    azdias.drop(outliers_list,axis = 1, inplace = True)

    #Dropping outliers from feat_info as well
    feat_info = feat_info[~feat_info.attribute.isin(outliers_list)]
    
    # select, re-encode, and engineer column values.
    type_dict={}

    for ind in feat_info.index: 
        row_type = feat_info['type'][ind]
        if row_type not in type_dict.keys():
            type_dict[row_type] = 1
        else:
            type_dict[row_type] += 1
            
    #Get a list of columns with categorial datatypes

    categorical_attribute = feat_info[feat_info.type == 'categorical'].attribute

    binary_cat, multi_cat = get_cat(azdias, categorical_attribute)
    
    #binary cat features re-encoding
    #find binary with non-numeric value
    for col in binary_cat:
        if azdias[col].dtypes == object:
            print(azdias[col].value_counts())

    #re-encoding non-numerical values
    azdias['OST_WEST_KZ'] = azdias['OST_WEST_KZ'].replace('W', 1)
    azdias['OST_WEST_KZ'] = azdias['OST_WEST_KZ'].replace('O', 0)
    print(azdias['OST_WEST_KZ'].value_counts())
    
    #Multilevel features will be dropped

    for col in multi_cat:
        azdias.drop(columns = col, inplace=True)
    
    #Finding mixed features
    mixed_features = feat_info[feat_info.type == 'mixed'].attribute
    
    # Create generation column
    azdias['PRAEGENDE_JUGENDJAHRE_decade'] = azdias['PRAEGENDE_JUGENDJAHRE'].apply(map_decade)

    # Create movement column
    azdias['PRAEGENDE_JUGENDJAHRE_movement'] = azdias['PRAEGENDE_JUGENDJAHRE'].apply(map_movement)
    
    # Create wealth column
    azdias['CAMEO_INTL_2015_wealth'] = azdias['CAMEO_INTL_2015'].apply(map_wealth)

    # Create life stage column
    azdias['CAMEO_INTL_2015_lifestage'] = azdias['CAMEO_INTL_2015'].apply(map_lifestage)
    
    # Drop PRAEGENDE_JUGENDJAHRE and CAMEO_INTL_2015 in addition to the rest of mixed features since we will not be using them
    for col in mixed_features:
        azdias.drop(columns = col, inplace=True)
        
    # Return the cleaned dataframe.
    
    return azdias
